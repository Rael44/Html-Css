Listas Ordenadas

Para criar uma lista ordenada, vamos usar a TAG(OL) para delimitar a lista. para identificar cada item da lista, usaremos (LI)

A tag (OL) possui varias formas, são eles :

1- Lista numerada
A- Lista alfabéticas maiúsculas (A)
a- Lista alfabéticas minúsculas (a)
I- Lista com algorismo Romanos maiúsculas (I)
i- Lista com algorismo Romanos minúsculas (i)

vejamos numericas:

  1. acordei
  2. levantei
  3. fui ao banheiro
  4. escovei os dentes

    vejamos romanas :

    1. acordei
    2. levantei
    3. fui ao banheiro
    4. escovei os dentes

      5. Listas não ordenadas

      A lista não ordenada é usada para marcar itens, pois a ordem dos itens não influênciam.

      Uamos a TAG (UL) de padrão a forma preenchida é um quadrado pintado

      1. DISC - bola preta pintada
      2. CIRCLE - bola com borda preta sem preenchimento(pintada)
      3. SQUARE - pequeno quadrado preto totalmente pintado
