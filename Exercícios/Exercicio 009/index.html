Usando links externos e interno

o são links internos?

Links internos é um link usado no seu site, o usuario entrará em outras abas ou menus,mas não saíra do seu site.

exemplos

voçê pode acessar o meu repositorio no GitHub

Voçê também pode acessar o meu YouTube

Links Externos

Oque são links externos

Links externos são os quais os usuarios saem do seu site clicando em algum link que leve ele até o Youtube,Facebook,Instagram e entre outros

Exemplos

Voçe pode acessar o meu repositorio no gitHub

voçê pode acessar o meu canal no youtube

-Nota-se que eu acrescentei 2 comandos para indicar o que deve ser feito

São eles

  • (target="_blank" rel="external")

Podemos ver que agora os links estão saindo da minha página

Site em outra pasta

Caso crie um site em outra página, o comando (<"a href=""> ) não irá voltar ou entrar, pois o meu outro site nao esta naquela pasta.

Só funcionará em sites de mesma pasta. Para resolver este problema devemos usar a seguinte TAG (../)

Esse sinal significa:voltar.

Então voltaremos pra pasta anterior e entraremos no site correto.
